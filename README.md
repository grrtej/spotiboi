# spotiboi

Emulate Spotify's premium features.

## Installation

### Dependencies

- Arch Linux: `gobject-introspection`
- Ubuntu/Debian: `pkg-config libgirepository1.0-dev libcairo2-dev`

### Install from git

`$ pip install -U git+https://github.com/grrtej/spotiboi.git`
